## 8. Stack Técnico Recomendado

### Opción A — WordPress (lanzamiento rápido)
- **CMS:** WordPress + GeneratePress + GenerateBlocks (o Elementor Pro)
- **Hosting:** Cloudways (DigitalOcean) o SiteGround
- **pSEO:** Custom Post Type + ACF Pro → plantilla dinámica con variables
- **SEO:** Rank Math Pro
- **Formularios:** WPForms → webhook a Make.com → notificación WhatsApp
- **Analytics:** GA4 + GSC
- **Speed:** WP Rocket + Cloudflare (dominio en CF)
- **CTA:** WhatsApp flotante (JoinChat) — número dedicado Web Sin Excusas
- **Email:** Resend para transaccional / Brevo para nurturing

### Opción B — Next.js + CMS Headless (más control)
- **Framework:** Next.js 14+ App Router
- **CMS:** Sanity.io (para gestionar municipios, sectores, portfolio)
- **Deploy:** Vercel
- **pSEO:** `generateStaticParams()` con datos de municipios/sectores en JSON/CMS
- **Forms:** Resend + webhook a Make.com
- **Analytics:** GA4 + Vercel Analytics
- **LLM routing:** Trovald (dogfooding para features inteligentes futuras)

### Recomendación
**WordPress para lanzar en 2-3 semanas**, con dominio en Cloudflare desde el día 1. La generación de landings pSEO con ACF + CPT es más rápida para un MVP. Migrar a Next.js cuando haya tracción y el volumen de landings justifique automatización total.

## 15. Notas Estratégicas — Web Sin Excusas

1. **pSEO ≠ contenido duplicado.** Cada landing debe tener mínimo 60% de contenido único. La variable `{excusa_local}` + datos reales del municipio hacen esto viable sin escribir cada página desde cero.

2. **Priorizar Tier 1 municipios.** Sabadell (sede), Terrassa, Mataró, Badalona tienen volumen de búsqueda real y menos competencia que "Barcelona" a secas. Atacar la capital desde las landings de sector.

3. **El recurrente es el objetivo real.** La web es la puerta de entrada (one-time), el SEO mensual es donde está el LTV. Los bundles Web + SEO deben ser el producto estrella en toda la comunicación.

4. **Google My Business propio.** Ficha optimizada como "Web Sin Excusas" en Sabadell. Fotos de la oficina/equipo, posts semanales, responder reseñas.

5. **Reseñas como motor.** Pedir reseña a cada cliente mencionando el municipio y servicio ("Nos hicieron la web para nuestro restaurante en Terrassa"). Esto refuerza E-E-A-T y las landings pSEO.

6. **WhatsApp > formularios.** En España, para PYMEs locales, WhatsApp convierte 3-5x más. El botón de WhatsApp debe ser omnipresente con un mensaje pre-rellenado: "Hola, he visto Web Sin Excusas y quiero información sobre..."

7. **El nombre es contenido.** "Web Sin Excusas" es un mensaje en sí mismo. Explotarlo en CTAs, títulos de blog, y social: "¿Aún sin web? Sin excusas.", "Tu negocio. Tu web. Sin excusas.", "La excusa de 'es muy caro' ya no vale."

8. **Cross-sell con Fast Horizons.** Los clientes existentes de Fast Horizons que encajen en perfil Barcelona pueden ser migrados/referidos. Web Sin Excusas es la marca pública para el mercado local; Fast Horizons sigue siendo la entidad legal y operativa.

9. **Contenido en castellano primero.** El target son dueños de negocios locales. Castellano como idioma principal, con opción de catalán en fase 2 para reforzar señal local.

10. **No competir por "agencia web barcelona".** Esa keyword es carísima y dominada por agencias con presupuesto. La estrategia es ganar por volumen de long tail: "diseño web [municipio]" × 50 municipios + "web para [sector] barcelona" × 30 sectores = 160+ landings que individualmente son fáciles de rankear.
